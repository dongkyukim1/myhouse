📑 기능정의서 (청약 공고 모니터링 & 준비 지원 웹서비스)
1. 서비스 개요

목적: 국민임대·공공임대·매입임대·공공분양 등 주택 청약 관련 정보를 한눈에 제공

대상: 20~40대 청년, 무주택 세대주, 생애최초 특별공급 대상자

주요 기능: 공고 수집 → 자격 검증 → 일정 관리 → 준비 서류 가이드

2. 핵심 기능
(1) 공고 수집·검색

SH공사, LH청약플러스, 청약홈 등 공공기관 사이트에서 공고 자동 크롤링

지역/유형별 필터: 서울/경기, 국민임대/매입임대/행복주택/분양

최신 공고 알림 기능 (푸시/이메일/SMS)

(2) 청약 자격 검증

사용자 기본정보 입력: 나이, 세대구성, 소득, 자산, 청약통장 납입횟수

조건 자동 대조 → “지원 가능 / 불가” 판정

지원 가능한 단지만 하이라이트 표시

(3) 일정 관리 (캘린더 뷰)

공고별 신청기간 / 서류제출 / 발표일 / 입주일 자동 등록

캘린더/타임라인 UI 제공

개인 캘린더(Google, iCal)와 연동

(4) 경쟁률/분석 제공

과거 동일 단지 경쟁률 데이터 시각화 (그래프/표)

예비 커트라인(점수, 납입횟수 등) 비교

사용자 점수 입력 시 → 당첨 가능성 시뮬레이션

(5) 서류 체크리스트

지원 유형별 준비서류 자동 안내

예: 주민등록등본, 가족관계증명서, 주민등록초본, 개인정보 수집동의서

제출처, 발급처, 온라인 발급 가능 여부 안내

체크박스 형태로 준비 상황 관리 가능

(6) 자금/대출 계산기

분양/임대 보증금 입력 → 계약금·중도금·잔금 납부 일정 자동 계산

대출(생애최초·중도금·주담대) 이자율 반영 → 월 상환액 시뮬레이션 제공

(7) 커뮤니티/후기

사용자 경험 공유 (예: “마장동 SH빌 실제 거주 후기”)

단지별 평점/리뷰 기능

3. 관리자 기능

공고 데이터 수동 등록/수정

크롤링 실패 시 수동 업로드 가능

사용자 통계(조회수, 신청 단지 수) 대시보드

4. 기술 스택 제안

Frontend: Next.js (React 기반), Tailwind CSS, Recharts (그래프)

Backend: Spring Boot or Node.js (API 서버)

DB: PostgreSQL (유저·공고·청약정보 저장)

크롤링: Python (BeautifulSoup, Selenium)

배포: Vercel(프론트), AWS EC2/RDS(백엔드·DB)

5. 사용자 플로우

회원가입 → 기본정보 입력(나이, 세대구성, 소득, 청약통장)

시스템이 자동으로 “지원 가능 단지 리스트” 제공

관심 단지 선택 → 일정 캘린더 자동 추가

서류 체크리스트 확인 + 준비 상황 기록

당첨 발표일 → 결과 입력/공유

6. 기대 효과

흩어진 공고 정보를 한눈에 모아 검색 피로도 ↓

자격/점수 자동 검증으로 지원 전략 수립 ↑

서류/일정 관리 자동화 → 청약 준비 효율 ↑ㅈ
